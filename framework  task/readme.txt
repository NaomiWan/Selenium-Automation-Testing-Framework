The HMS test automation framework

This is a hybrid selenium framework for the automated testing of a Hospital management web application. To run the scripts in this framework, you need to have the latest version of eclipse installed. you also need to have the appropriate drivers for the web browsers you intend to use. this framework has considered FireFox Mozilla, Chrome and Internet Explorer browsers.

Installing and runing the scripts

-download the zipped project folder from GIT
-extract the project files to a location on your computer
-import the project files into eclipse on your local machine
-open the "Constants.Java" file under the "config" package in the "src" folder and change the respective driver path for the browser you'll be using.
-Under the "tests" package in the "src" folder, open the "TestsDriver.java" file and run it.
-you can view the results of the test under test-output folder
